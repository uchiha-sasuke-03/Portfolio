# 🍥 KAS.Shinobi Portfolio - The Path of the Developer

[![Naruto Theme](https://img.shields.io/badge/Theme-Naruto%20%2F%2090s%20Pop%20Anime-orange)](https://github.com/uchiha-sasuke-03/Portfolio)
[![Tech Stack](https://img.shields.io/badge/Stack-React%20%2B%20Vite%20%2B%20Tailwind-blue)](https://github.com/uchiha-sasuke-03/Portfolio)

Welcome to my digital ninja way! This is a high-energy, immersive portfolio inspired by the 90s Naruto aesthetic, blending traditional shinobi vibes with modern Full-Stack & AI technologies.

## 🥷 Features

- **Dattebayo! Hero Section**: 
  - Dynamic **3D Card Flip** between profile photo and Naruto character.
  - Floating **Shinobi Badges** (Chidori Speed, Will of Fire, etc.) with parallax effects.
  - Custom **Hidden Leaf Village** branding.
- **Drifting Nindo (Quotes)**: Immersive Naruto quotes that drift across the background from right to left, spread throughout the entire site with random staggered timing.
- **Jonin-Rank UI**: 
  - **90s Pop Anime Aesthetics**: Halftone patterns, speed lines, and vibrant ink-splatter effects.
  - **Thematic Typography**: Using *Russo One* for bold Shonen-style headings and *Nunito* for readable body text.
- **Ninja Arts (Interests)**: A dedicated section featuring **Madara Uchiha** and personal hobbies.
- **Responsive Jutsu**: Fully optimized for all screen sizes, from mobile scrolls to desktop displays.

## 🛠️ Tech Stack (The Ninja Tools)

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS, Vanilla CSS (for speed lines & halftone)
- **Animations**: Framer Motion (3D Flips), GSAP (Scroll-linked parallax), Anime.js (Impact entrances)
- **Icons**: Lucide React

## 🚀 Deployment & Installation

1. **Clone the Scroll**:
   ```bash
   git clone https://github.com/uchiha-sasuke-03/Portfolio.git
   ```

2. **Gather Chakra (Install Dependencies)**:
   ```bash
   npm install
   ```

3. **Release the Jutsu (Run Locally)**:
   ```bash
   npm run dev
   ```
